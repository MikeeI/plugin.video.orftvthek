ORF TVthek KODI Addon
=======
ORF TVthek is an addon that gives you access to the ORF TVthek Video Platform.


Requirements
------------
Tested on XBMC Frodo/Gotham and KODI 14.2


Supported platforms
-------------------
Windows, Linux , Android and OSX


Current Features
----------------
* Livestream
* All Shows
* Schedule Search
* HTTP Stream H264 (Stable)
* Search Function
* Missed Shows
* Blacklist Shows
